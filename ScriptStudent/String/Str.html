Строки

Для создания строк мы можем как напрямую присваивать переменной строку: let name = "Tom"; Для работы со строками предназначен объект String, поэтому также можно использовать конструктор String: var name = new String("Tom"); Но как правило, используется первый более краткий способ. В первом случае JavaScript при необходимости автоматически преобразует переменную примитивного типа в объект String. Объект String имеет большой набор свойств и методов, с помощью которых мы можем манипулировать строками.
