Свойство flex-flow позволяет установить значения сразу для обоих свойств flex-direction и flex-wrap. Оно имеет следующий формальный синтаксис:

flex-flow: [flex-direction] [flex-wrap]

Свойство order позволяет установить группу для flex-элемента,

Flex Item 1
order 0
Flex Item 2
order 1
Flex Item 3
order 1
Flex Item 4
order 0
Flex Item 5
order -1

элементы в группе 0 располагаются перед элементами с группой 1, а элементы с группой 1 располагаются перед элементами с группой 2 и так далее.
