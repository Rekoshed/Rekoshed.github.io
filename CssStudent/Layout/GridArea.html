Наложение элементов

Благодаря установке свойства z-index: 1 у всех элементов получается, что как первый элемент как-будто лежит фоном под другими пятью элементами.

При необходимости слой из первого элемента можно наоборот перенести ближе к пользователю, накрыв остальные элементы. Для этого надо увеличить у него значение z-index: 


      .item1{
      grid-area: 1 / 1 / 3 / 4;
      opacity: 0.8;
      z-index: 10;
      }
