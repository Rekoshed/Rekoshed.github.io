Синтаксис CSS

CSS состоит из правил стиля, которые браузер интерпретирует и затем применяет к соответствующим элементам в вашем документе. Каждое правило имеет три части: селектор, свойство, и значение.

Например, цвет заголовка может быть определен как:

h2 {color:red;}

где h2 - селектор, color - свойство, red - значение. Стилевые свойства перечисояются через точку с запятой и заключаются в фигурные скобки, если речь идет о стилевом блоке.

селекторы делятся на три основные группы:

  • Селекторы по тегу

  • Селекторы по классу

  • Селкторы по идентефикатору

Но также могут быть:

  • Группы селекторов
