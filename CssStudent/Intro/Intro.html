Есть три вида стиля веб страницы:

  • Стиль браузера.Задаётся браузером по умолчанию.Голый html. Имеет низший приоритет.
  • Стиль Автора.Задаётся разработчиком.
  • Стиль пользователя. Задаётся пользователем в настройках браузера. Имеет высший приоритет.

CSS означает Каскадные Таблицы Стилей (Cascading Style Sheets).
