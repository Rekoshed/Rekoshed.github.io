Вот несколько вариантов установить Python. Python можно скачать и установить с официального сайта Python, но в приоритете конечно выполнить эту процедуру через пакетные менеджеры:

Ubuntu

$ sudo apt update
                    $ sudo apt install python3

macOS

# https://brew.sh/index_ru.html
                    $ brew install python

Windows

# https://chocolatey.org/
                    $ choco install python

Убедитесь в том, что Python работает. Наберите в терминале python3 --version. Вывод должен быть примерно такой:

                            rekoshed@rekoshed:~$ python3 --version
                            Python 3.7.3

Для запуска кода необходим файл с любым именем и расширением py .

    
            $ touch hello.py                       # создадим файл hello.py 
            $ echo 'print("hello")'>> example.py   # запишем туда простой код
            $ python3 hello.py                     # запустим код в итерпретаторе

Всё просто. Если ошибок нет, то интерпретатор выведет строку приветствия в командной строке терминала
