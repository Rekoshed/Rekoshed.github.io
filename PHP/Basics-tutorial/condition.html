Инструкция if else используется для проверки условия. Есть различные способы использования иструкций if

if

If

if выполняется если условие верно.

if (condition){
//code to be executed
} 

	  Пример:
	  
	  <?php  
	  $num=12;  
	  if($num<100){  
	  echo "$num is less than 100";  
	  }  
	  ?>
	  
	  Output:
	  12 is less than 100
